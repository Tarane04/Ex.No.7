### **1. Aim**

The goal of this experiment is to create a **Prompt-Based AI Application** using ChatGPT that functions as a **Personal Productivity Assistant**. It should manage daily tasks, reminders, wellness tips, and adapt to user preferences.

---

### **2. Why We Do This Experiment**

This lab helps you understand:

* How to design prompts for real-life applications
* How AI can assist in task management
* How prompts can evolve from simple → advanced
* How an AI can behave like a smart assistant

---

### **3. Tools Required**

* **ChatGPT / Any Large Language Model (LLM)**
* Optional: Memory-enabled version (so it can learn preferences)
* A simple interface (chat window / Python CLI / web UI)

These are enough to build the assistant **without coding** initially.

---

### **4. Explanation of the Concept**

We are building a **productivity assistant** only using prompts.
This assistant should:

| Feature              | What it Should Do                     |
| -------------------- | ------------------------------------- |
| Task Manager         | Add, store, edit, and manage tasks    |
| Scheduler            | Set reminders and avoid clashes       |
| Wellness Tips        | Give health and self-care suggestions |
| Learning Preferences | Improve tips based on feedback        |

We achieve this by creating **structured prompts** that guide the AI’s behavior.

---

### **5. System Workflow (How It Works)**

The user interacts with the assistant in simple language.
The assistant processes the input, understands it, gives responses, and (optionally) learns from the user's feedback.

**Simple Flow:**

1. User gives an input (e.g., “Remind me to study at 7 PM”)
2. LLM processes the request
3. Assistant replies and performs the task
4. User gives feedback (optional)
5. System adapts to improve next response

---

### **6. Procedure (Step-by-Step)**

Follow these steps to perform the experiment:

#### **Step 1: Identify the Core Features**

Decide what functions the assistant must perform:

* Task management
* Scheduling
* Wellness guidance
* Answering general questions

#### **Step 2: Create Prompts for Each Feature**

Example prompts:

| Stage               | Example Prompt                                  |
| ------------------- | ----------------------------------------------- |
| Basic Prompt        | “List my tasks for today.”                      |
| Intermediate Prompt | “Organize my tasks based on urgency and time.”  |
| Advanced Prompt     | “Remember that I prefer workout tips at night.” |

#### **Step 3: Simulate a User Interaction**

Act like a user and chat with the AI:

**User:** Remind me to drink water every 2 hours.
**Assistant:** Reminder added – I will remind you every 2 hours.

#### **Step 4: Collect Feedback from User**

Example:
**Assistant:** “Would you like more mental-health tips or exercise tips?”
User feedback helps improve suggestions.

#### **Step 5: Add Memory (Optional)**

This step makes the assistant smarter.
If user says: “I don’t like yoga, give gym-based tips,”
the assistant stores this and avoids suggesting yoga again.
